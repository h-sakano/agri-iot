# 概要

庭の野菜ちゃんずの管理が行き届かなくなってきたのでIoT化を行う。

# 機材

+ Raspberry Pi 3 Model B
+ DHT11(温湿度センサー)
+ YL-69(土壌水分センサー)
+ ADC0832(A/Dコンバーター)
+ ジャンパワイヤー
+ microSDカード(AUSDH16GUICL10-RA1)
